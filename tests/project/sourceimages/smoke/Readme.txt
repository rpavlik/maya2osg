This directory contains a default set of source images for the 
dynamics smoke effect (Dynamics-->Effects-->Create Smoke).  

To use these images,

1) Ensure that your current project has a directory set for
   Source Images box. (File-->Project-->Edit Current)
2) Copy the images to the directory specified in Source Images
   box.
3) Enter the name "Smoke.0" in the Sprite Image Name box of
   the Create Smoke option box.

--------------------------------------------------------------

This directory also contains a convenience script (applySprite.mel)
for applying any set of sprites to any particle object.  
See the header section of the script for description and usage.
